# WuTang Packet Tracer Labs

Daily 20-minute Cisco Packet Tracer labs focused on building real networking skills through consistency, repetition, and practical implementation.

---

## 🎯 Objective

The goal of this repository is to:
- Build strong networking fundamentals from the ground up
- Develop hands-on experience with switching, routing, and network services
- Maintain daily consistency with low-friction 20-minute labs
- Document progress for learning, reflection, and portfolio purposes

---

## ⚙️ Lab Structure

Each lab follows a simple structure:

- ⏱️ Duration: 20 minutes (hard stop)
- 🧪 Focus: One core networking concept
- 📝 Documentation: Short README per lab
- 📁 Files: Packet Tracer topology included

## 🧠 Topics Covered

- Basic Networking (PC-to-PC communication)
- Switching Fundamentals (MAC tables, VLANs)
- Routing (Static, Inter-VLAN)
- IP Addressing & Subnetting
- DHCP & Network Services
- Network Security (Port Security, ACLs)
- Troubleshooting

---

## 🚀 Approach

This project is built on:
- Consistency over intensity
- Practical labs over theory-heavy learning
- Real-world scenarios over perfect configurations

## 📘 Why This Matters

Hands-on networking is the fastest way to:
